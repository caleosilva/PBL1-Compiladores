Ordem de identificação:

1. Comentários (comments)
2. Strings ():
    - Todo o conteúdo entre aspas duplas, mesmo sendo palavras reservadas, deve ser considerado parte da string.
3. Caractere (character)
    - Tudo que estiver entre aspas simples, mesmo que signifique outra coisa, deve ser considerado parte do mesmo caractere.
    - OBS: Só pode ter um caracterer dentro das aspas simples, caso contrário deve ser um erro.
4. Delimitadores (delimiters)
    - Nos casos de (), {} e [], sempre deve abrir e fechar, caso contrário é um erro.
5. Numeros (numbers)
6. Operatores
7. Identificadores (identifiers)
8. Palavras chave (key_words)




Exemplo:

class ExampleClass extends BaseClass {
    variables intVar = -3-9.7;
    ExampleClass.falou("Meu nome /*eh*/ class")
}

Deve ter como saída:

keywords: [class, extends, variables]
identifiers: [ExampleClass, BaseClass, intVar, falou]
operators: [=, -]
numbers: [-3, 9.7]
delimiters: [{, ;, ., (, ), }]
strings: ["Meu nome /*eh*/ class"]
